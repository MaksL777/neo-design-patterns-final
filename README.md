# Фінальний проєкт: Генератор Резюме

## Реалізовані патерни
1. **Facade (Фасад):** Клас `ResumePage` приховує складність виконання HTTP-запиту, парсингу JSON та ініціалізації процесу рендерингу. Клієнт (`main.ts`) викликає лише один метод `init()`.
2. **Template Method (Шаблонний метод):** Клас `AbstractImporter` задає жорстку послідовність кроків `process()`: `validate` -> `map` -> `render`. Клас `ResumeImporter` реалізує ці кроки.
3. **Factory Method (Фабричний метод):** Клас `BlockFactory` інкапсулює логіку створення конкретних блоків (Header, Summary тощо). Він приймає рядок-тип і повертає об'єкт, що реалізує інтерфейс `IBlock`.
4. **Composite (Компонувальник):** `ExperienceBlock` (контейнер) містить у собі масив `ProjectBlock` (листя). Під час свого рендерингу контейнер рекурсивно викликає метод `render()` у своїх нащадків.
5. **Decorator (Декоратор):** Клас `HighlightDecorator` обгортає готовий `ProjectBlock`. Він розширює його поведінку (додає CSS-клас `.highlight` для `isRecent: true`), не змінюючи внутрішнього коду самого блоку проєкту.
